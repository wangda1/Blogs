## 个人感悟  

初步学习一个领域的知识时，不要纠结于细节的实现，这会让你抓不到大观，也不建议开始啃大部头书，这会打击学习兴趣  
个人感觉最佳学习路线应该是：先明白做什么 --> 再明白原理与原因 （为什么）--> 后学习实现细节 --> 最后尝试修改实现来优化效果
