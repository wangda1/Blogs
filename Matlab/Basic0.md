# Start here for Matlab  

## 常用命令总结   
> - clc 清零命令窗口  clear 清零变量窗口  
> -   









## 函数常用命令  
> - type 可用于查看函数的定义  type('sphere') 或 type sphere   
> - global 用于定义全局变量  global G  
> - nargin 获取输入变量个数  nargout 获取输出变量个数  varargin 用于不确定输入变量的输入  
> - pathtool 用于设置函数搜索路径  addpath  添加新的搜索路径
